# 🚀 Alumni-connect
Alumni-connect is Bridge Between Students & Alumni, Student get a new Opportunities from the trusted Source.

---

## 🧠 About the Project
- Alumni Connect is Bridge Between Students & Alumni, Student get a new Opportunities from the trusted Source.
- Where Alumni can post a new opportunity for the student and student can apply on it,
- When New Post arrived than all connected student's got a new Email Notification about the new post,
- Simple Connection mechanism are there for the simply student and alumni are connect with each other.

---

## ⭐ Features
- 🔐 Authentication ( RBAC System )   
- 📊 Dashboard ( Admin )
- 👤 User data list ( Admin )
- 📝 Post upload ( Alumni )
- ✅ Apply on post ( student )
- 📧 Email Services

---

## 🛠 Tech Stack
- **Frontend:** HTML / CSS / JS / BootStrap
- **Backend:** CORE PHP
- **Database:** MySQL  
- **Other:** PHPMailer
